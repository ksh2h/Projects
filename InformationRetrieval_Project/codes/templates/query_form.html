Query Formats

  • Queries should be entered in English language.
  • There are 7 distinct types of queries which are processed by this software ( please refer to the Documentation )
  • Format for entering author name : ['author', 'author name', 'by', 'author is'] <author's name>
  • Format for entering venue/conference name : ['at', 'venue', 'conference', 'venue is', 'conference is'] <venue/conference name>
  • Format for entering field name : ['in', 'based on', 'field', 'field is'] <field name>
  • Format for entering co author name : ['coauth of'] <author name> OR <author name>'s coauth
  • Format for entering year : ['between', 'from'] <start year> ['and', 'to'] <end year> OR ['till'] <end year>
  • For citation count and publication count include the keywords "citation" and "publication" in query
  • [.] in the above formats represents any one of these alternatives should be used
  • After each filter & sign should be placed.
